# Bellwether Typography Reference

## Font Pairing

Bellwether documents use a consistent two-font system across Microsoft Office:

- **Garamond** — document and slide titles (serif, authoritative, warm)
- **Avenir Next LT Pro** — body text, bullets, labels, captions (sans-serif, clean, readable)
  If Avenir Next LT Pro is unavailable, substitute **Aptos**.

---

## Word Documents

### Heading Levels

| Level | Font | Weight | Size | Color | Spacing after |
|---|---|---|---|---|---|
| Document title | Garamond | Bold | 24pt | Black | 12pt |
| H1 (major section) | Avenir Next LT Pro | Bold | 16pt | Plum | 10pt |
| H2 (subsection) | Avenir Next LT Pro | Italic | 12pt | Plum | 8pt |
| H3 (minor section) | Avenir Next LT Pro | Bold | 11pt | Black | 6pt |

### Body Text

| Role | Font | Weight | Size | Line spacing |
|---|---|---|---|---|
| Body paragraph | Avenir Next LT Pro | Regular (45) | 11pt | 1.15 |
| Caption / footnote | Avenir Next LT Pro | Regular (45) | 9pt | 1.0 |
| Pull quote / callout | Avenir Next LT Pro | Medium (65) | 11pt | 1.15 |
| Table header | Avenir Next LT Pro | Medium (65) | 11pt | 1.0 |
| Table body | Avenir Next LT Pro | Regular (45) | 11pt | 1.0 |

### General Rules for Word Docs
- Use the provided letterhead templates rather than creating documents from scratch.
  The templates have correct heading styles, margins, and header/footer placement.
- Body text color: standard black (`#000000`)
- Separate body paragraphs with a blank line. Do not run paragraphs together without any visual break.
- Avoid underlining for emphasis — use bold or italic instead.
- Do not use decorative fonts or mix in additional typefaces.

## PowerPoint Presentations

**Note**: Bellwether slides for external presentations (e.g., conferences) 
  should use less text and more visual elements. Slides for documentation 
  purposes (e.g., client deliverables or factbases) may have more text and 
  citations. If the deck's purpose is unclear, ASK the user.

### Slide Text

| Role | Font | Weight | Size |
|---|---|---|---|
| Slide title | Garamond | Bold | 28–36pt |
| Section divider headline | Garamond | Bold | 40–48pt |
| Subtitle / deck title | Garamond | Regular | 20–24pt |
| Body text / bullets (L1) | Avenir Next LT Pro | Regular (45) | 16-18pt |
| Body text / bullets (L2) | Avenir Next LT Pro | Regular (45) | 12-16pt |
| Data labels | Avenir Next LT Pro | Medium (65) | 12–14pt |
| Source / footnote | Avenir Next LT Pro | Regular (45) | 10pt |
| Callout box text | Avenir Next LT Pro | Medium (65) | 14–16pt |

### General Rules for Presentations
- Always start from `Extended PPT Deck Template 2024.potx` — it contains the
  correct master layouts, font assignments, and color theme.
- Keep slide titles concise (headline-style, not full sentences).
- Lead each slide with the key finding or takeaway in the title — don't make
  the audience dig into the body to understand the point.
- Limit body bullets to 6-8 per slide. Prefer shorter, parallel constructions.
- One idea per slide is the ideal; consolidate only when content is genuinely
  supplementary.
- Do not add contact information or modify the content of closing or Thank You slides.

## Spreadsheets / Excel

Text sizing in spreadsheets is typically smaller than Word or Powerpoint — use:

| Role | Font | Weight | Size |
|---|---|---|---|
| Sheet title / report header | Garamond | Bold | 14–16pt |
| Column headers | Avenir Next LT Pro | Medium (65) | 10pt |
| Data cells | Avenir Next LT Pro | Regular (45) | 10pt |
| Totals / summary rows | Avenir Next LT Pro | Medium (65) | 11pt |
| Footnotes | Avenir Next LT Pro | Regular (45) | 9pt |

If Avenir Next LT Pro is not available in Excel, use **Aptos** throughout.

## Avenir Next LT Pro Weight Guide

| Weight name | Number | When to use |
|---|---|---|
| Ultra Light | 25 | Decorative display only — rarely needed |
| Light | 35 | Large display text with plenty of white space |
| Regular | 45 | Standard body text, bullets |
| Medium | 65 | Emphasis, table headers, callout text |
| Demi | 75 | Strong emphasis, subheadings in PPT |
| Bold | 85 | High-contrast labels, rarely needed alongside Garamond |
